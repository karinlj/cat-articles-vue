# cat-articles

## Description

Application for exploring Vue.js. CRUD and jsonPlaceholder.

## Prerequisites

NodeJS

## Installation

To clone
`git clone https://github.com/karinlj/cat-articles-vue.git.`

```
npm install
```

## Compiles and hot-reloads for development

```
npm run serve
```

## To build

```
npm run build
```

Then use the dist folder you get.

## Built With

- vue.js
- CSS
- Materialize.css
- jsonPlaceholder
